Understanding Your Scores

IBLTS provides scores on a 9-band scale for each section of the test as well as an overall band score. Here's what each band level represents:

Band Score Skill Level Description
9 Expert user Fully operational command of the language
8 Very good user Fully operational command with occasional inaccuracies
7 Good user Operational command with occasional inaccuracies
6 Competent user Effective command despite some inaccuracies
5 Modest user Partial command, coping with overall meaning
